You are working on the NestJS preview server in `preview-server/`.

## Rules

- Follow the existing TypeScript and NestJS patterns in this repo
- Use single quotes and semicolons
- Keep controllers, services, and modules straightforward and cohesive
- Prefer minimal, targeted changes over broad refactors

## Verification

Run from `preview-server/` when code changes are made:

```bash
npm run build
```

Manually verify with:

```bash
npm run start
```
